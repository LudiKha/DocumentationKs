LowLevel

"com.kinstrife.core.low-level": "file:/GameDev/Kinstrife/Packages/Core/LowLevel"

 

LowLevel is a (WIP) foundational package in the Kinstrife Core suite that provides common low-level runtime functionalities, as an extension of UnityEngine.LowLevel.

This package provides several essential runtime systems and types that other packages build upon. It includes:

  • Player Loop Control: Manage the main game loop, allowing for custom update cycles and frame management.
  • Time: Timer utilities for scheduling tasks and measuring elapsed time.
  • Caching Utilities: Tools for caching frequently used data such as Type or Unity.Object names to improve performance.
  • Application Control: Utilities for managing application state and lifecycle events.
  • Thread-compatible APIs: Alternatives to Unity's main thread-only APIs, enabling multi-threaded updates.
