LODComponents

Holds references to components that should be enabled or disabled based on LOD distance.

LODController

Manages LOD distance and visibility-based activation of associated LODComponents.

  1. Configure Kinstrife.LOD.LODController.components to assign different components to specific LOD distances.
  2. Integrate with a culling system that provides callbacks. (provided in Kinstrife.Systems packages.)
  3. When a visibility or distance change occurs, call StateChanged(CullingGroupEvent) with the event data.
  4. The controller will automatically enable or disable the relevant components based on the current LOD distance and visibility state.
  5. The event is propagated via OnStateChange for further custom handling.
