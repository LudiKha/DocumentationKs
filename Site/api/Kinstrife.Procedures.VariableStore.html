Class VariableStore

Manages the storage, retrieval, and application of variable definitions and their instances.

VariableStore
Namespace: Kinstrife.Procedures

public sealed class VariableStore : MonoBehaviour
Remarks

The VariableStore class provides functionality to manage variable definitions and their instances, including retrieving variables by key, applying new definitions, and filtering variables based on their types. It supports both immediate and deferred variable instances, and maintains internal caches for efficient access.
